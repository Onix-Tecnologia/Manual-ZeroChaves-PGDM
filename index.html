Zero Chaves PGDM

Introdução

O Zero Chaves PGDM é uma solução inovadora para portas giratórias com detecção de metais, eliminando a necessidade de chaves físicas para o travamento e destravamento das portas.

O sistema oferece controle de acesso por reconhecimento facial, proporcionando um nível superior de segurança e permitindo a geração de relatórios detalhados de acessos. Além disso, a solução conta com um avançado sistema antiburla, capaz de detectar tentativas de entrada forçada e acionar alarmes via rede de forma automática.

Funcionamento

Quando destravado, o sistema opera de forma semelhante a outras portas giratórias com detecção de metais, mas se destaca por eliminar o uso de chaves físicas no travamento e destravamento. Esse processo é realizado pelo controlador de acesso conectado diretamente à porta.

Uma vez que a porta é travada, o sensor de detecção de tentativa de entrada forçada é ativado. Caso uma tentativa de intrusão seja identificada, um alarme é automaticamente gerado e transmitido pela rede.
